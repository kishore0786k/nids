# Compact IEEE Paper Skeleton

## Title

Confidence-Aware Neuro-Symbolic Intrusion Detection for IoT NetFlow Traffic

## Abstract Claim

We propose a neuro-symbolic NIDS that combines an MLP classifier with data-calibrated symbolic rules. Unlike passive rule overlays, the proposed layer uses training percentiles and prediction probabilities to perform confidence-aware hard override or soft fusion. Experiments on NF-ToN-IoT-V2 report baseline metrics, rule trigger/change rates, hard/soft ablations, confusion matrices, false-negative counts, and per-class recall/F1 deltas.

## Contributions

1. A data-driven symbolic rule engine calibrated from training-distribution percentiles.
2. Confidence-aware hard and soft neuro-symbolic fusion using neural probability vectors.
3. Targeted attack false-negative reduction for benign-looking attack traffic.
4. A reproducible experiment runner with ablations, rule diagnostics, per-class reports, and confusion matrices.

## Experimental Protocol

Use `data/train_processed.csv` for model/rule calibration and `data/test_processed.csv` for final testing. Compare RandomForest, MLP, MLP without rules, hard neuro-symbolic fusion, and soft neuro-symbolic fusion. Report accuracy, macro precision/recall/F1, per-class metrics, confusion matrix, rule trigger rate, prediction-change rate, and attack false negatives before/after rules.

The reproducibility artifact is generated by:

```bat
venv\Scripts\python.exe -m src.experiment_runner --fusion-mode soft --alpha 0.65
```

For robustness evidence, run:

```bat
venv\Scripts\python.exe -m src.experiment_runner --seeds 42,43,44
```

## Recommended Claim Boundary

Do not claim state-of-the-art overall accuracy unless external baselines are added. Claim improved MLP detection quality and auditable false-negative reduction for selected attack classes.
