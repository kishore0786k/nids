# Neuro-Symbolic NIDS Paper Package

This folder contains generated IEEE-style research artifacts for the NF-ToN-IoT-V2 neuro-symbolic intrusion detection project.

## Generate Artifacts

Run from the project root:

```powershell
.\build_publication_package.bat
```

or:

```powershell
.\venv\Scripts\python.exe -m backend.generate_publication_package
```

## Journal Upgrade Experiment

The XGBoost + Isolation Forest cross-dataset experiment is generated by:

```powershell
.\venv\Scripts\python.exe -m src.journal_nids_upgrade
```

For a quick artifact smoke test:

```powershell
.\venv\Scripts\python.exe -m src.journal_nids_upgrade --smoke
```

For a capped but less order-biased dry run, add deterministic CSV sampling:

```powershell
.\venv\Scripts\python.exe -m src.journal_nids_upgrade --sample-mode random --max-train-rows 20000 --max-external-rows 10000
```

Outputs are written to:

- `models/journal_upgrade/`: XGBoost, Isolation Forest, scaler, label encoders, and feature schema.
- `results/journal_upgrade/journal_upgrade_manifest.json`: run configuration, metrics, and artifact manifest.
- `paper/generated/journal_upgrade/`: four core paper tables in CSV, TeX, and Markdown form.
- `paper/figures/journal_upgrade/`: eight core paper figures as PNG and PDF, plus SHAP beeswarm/waterfall supplements.

## Outputs

- `paper/generated/*.tex`: abstract text, setup table, comparison table, algorithm outline, and results paragraph.
- `paper/figures/generated/*.png`: high-resolution figure previews.
- `paper/figures/generated/*.pdf`: publication figure files.
- `results/publication_package/publication_package.json`: reproducibility artifact.
- `results/publication_package/publication_summary.md`: compact results summary.
- `results/publication_package/submission_readiness.md`: remaining manual checks.
